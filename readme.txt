Las pruebas requeridas fueron realizadas con Cypress, se encuentran en la ruta "./cypress/e2e"

El archivo "addToCart.spec.cy.js" ejecuta la prueba e2e para añadir los productos al carrito y proceder a la compra.

El archivo "apiTesting.spec.cy.js" ejecuta los comandos para agregar y modificar la información de la mascota.

Instrucciones de ejecución

Node version > 18

1. Ejecutar en consola el comando "npm install"
2. Abrir Cypress con el comando "npx cypress open"
3. Escoger la opcion "E2E Testing"
4. Escoger navegador
5. Pestaña "Specs"
6. Ejecutar archivos

Además se encuentran videos demostrando como funciona cada prueba en la carpeta "Resources".
